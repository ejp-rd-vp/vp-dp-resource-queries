Query the VP Resource Interface (queryV1Endpoint)
Muliple parameters can be provided. Whereas the only mandatory parameter is the disease code.

Query parameters

disease (required)
Query Parameter — Orphanet Rare Disease Code
token (optional)
Query Parameter — Authorization Token
types (optional)
Query Parameter — Resource type
countries (optional)
Query Parameter — Location of the resource
genders (optional)
Query Parameter — Patient gender
minAge (optional)
Query Parameter — Patient minimum age
maxAge (optional)
Query Parameter — Patient maximum age

array[SourceResponse]
